KeyStoreLite Specifications

Specification

Name: KeyStoreLite.

Function: Save data to device storage.

Component: FileInputStream, FileOutputStream, FileWriter, and Application Context.

Ways of working: Initialize > save > saved to device storage or RAM > read > data can be reused.

Storage capacity: 2 MB per file for keystore.external.local, 1.5 MB for keystore.internal.local

Keys that can be saved: 35 for keystore.external.local, 25 for keystore.internal.local, 30 for keystore.session.store

Files that can be created: 4 for keystore.external.local, 3 for keystore.internal.local

Compatibility: Java 7 or later.
