KeyStoreLite Update 1.2.5

KeyStoreLite update version 1.2.5, but the features are not as many as the previous update. Now, you can activate 3 arrays at once.

You can create 3 object instances of the Array class.

Array user_data = new Array(this, "user_data");
Array user_stats = new Array(this, "user_stats");
Array user_logs = new Array(this, "user_logs");

user_data.addInt();
user_stats.addInt();
user_logs.addString();

Notes

If you create more than 3 instances of Array objects, KeyStoreLite will throw an IllegalStateException.
