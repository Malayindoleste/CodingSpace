KeyStoreLite Update 1.4.0

Now you can manage data that is in different files at one time. Just like Array, you simply create an instance of the Store class object to use its methods.

Store user_data = new Store(this, "user_data");
user_data.addString(key, value);
user_data.addBoolean(key, value);
user_data.addInt(key, value);
user_data.addFloat(key, value);

user_data.getString(key);
user_data.getBoolean(key);
user_data.getInt(key);
user_data.getFloat(key);

For methods like createFile(), you no longer need that method because the file will be automatically created when you create the Store object.

user_data.createFile(filename); // This is no longer necessary
user_data.delete(key);
user_data.clearFile(); // No need to mention the file name
