KeyStoreLite Update 1.2.0

KeyStoreLite update

We have added support for storing data in array form. Arrays will be stored in .ary files, but the format will remain as json.

How to use

Import the class first.

import keystore.array.external.AryStore;
import keystore.array.internal.AryStore;

After importing the class, initialize it first.

AryStore.init(this);

Then activate the array.

AryStore.setArray("user_data");

Once these three steps are complete, you can save, read, and delete data from the array.

Example of storing data at index 0.

AryStore.addString("Hello world!", 0);

Example of reading data from a specific index.

AryStore.getString(0):

Example of deleting data from a specific index.

AryStore.remove(0)

To clear the array.

AryStore.clear();

Complete method

To save data.

AryStore.addString(value, index);
AryStore.addInt(value, index);
AryStore.addBoolean(value, index);

To read data.

AryStore.getString(index);
AryStore.getInt(index);
AryStore.getBoolean(index);

To read all data

AryStore.getAllString(startIndex, endIndex);

To delete data.

AryStore.remove(index);

To clear the array.

AryStore.clear();

Notes

.ary files can only store arrays up to 30 indexes. If the limit is exceeded, KeyStoreLite refuses to store new data.

.ary files can only be created once. If setArray() is called twice with different parameters, the second array cannot be created or activated. But the old array must be activated in another activity (if you work on android).

If you skip the index directly, an empty index will be assigned a default value ("").

The "startIndex" and "endIndex" parameters are parameters used to read data from a certain index to a certain index (for example, reading data from index 0 to index 3, data stored at index 3 and above will be ignored). Make sure the method to read all data is called on String[]/int[]/boolean[] array = AryStore.getAllString/Int/Boolean()
