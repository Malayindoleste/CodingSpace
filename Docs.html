KeyStoreLite - Official Documentation

Introduction

KeyStoreLite is a simple key-value storage library for Java that works similarly to localStorage and sessionStorage in JavaScript. Data is stored in JSON files in internal or external storage, with a very easy-to-use interface.

How to Use

Add KeyStoreLite1.1.0-beta.jar to your project's libs/ folder, then import the classes:

import keystore.external.local.Store;
import keystore.internal.local.Store;
import keystore.session.store.Session;

After importing, initialize KeyStoreLite first using the method:

Store.init(this);
Session.init(this);

Example of creating a new file:

Store.createFile("user_data.json");

Example of adding an item:

Store.addString("level", "10", "user_data.json");
Session.addString("sessionToken", "is_login");

Example of reading an item:

String level = Store.getString("level", "user_data.json");
String sessionToken = Session.getString("sessionToken");

Complete Methods

To add data.

Store.addString(key, value, destinationFile);
Store.addInt(key, int, destinationFile);
Store.addBoolean(key, boolean, destinationFile);

Session.addString(key, value);
Session.addInt(key, int);
Session.addBoolean(key, boolean);

To read data.

Store.getString(key, destinationFile);
Store.getInt(key, destinationFile);
Store.getBoolean(key, destinationFile);

Session.getString(key);
Session.getInt(key);
Session.getBoolean(key);

To delete data.

Store.delete(key, destinationFile);
Session.remove(key);

To clear all stored data.

Store.clearFile(targetFile);
Session.clearSession();

Class function

import keystore.external.local.store.Store;

This class functions to save data to external storage

/sdcard (or storage/emulated/0/)/Android/data /package.name/files/

import keystore.internal.local.store.Store;

This class functions to save data to internal storage

/data/data/package.name/files/

keystore.session.store.Session;

This class functions to save data to memory

Will be deleted if the application is closed

Notes

If a file is deleted using clearFile(), it cannot be recovered (but can be recreated empty).

Data stored in Session.class will be automatically deleted when the app is closed or when the system needs memory.

To see the KeyStoreLite specifications, can visit KeyStoreLite Specifications

KeyStoreLite is not suitable for large-scale applications.
