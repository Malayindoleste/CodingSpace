KeyStoreLite Specifications

Specification

Name: KeyStoreLite.

Function: Save data to device storage.

Component: FileInputStream, FileOutputStream, FileWriter, and Application Context.

Ways of working: Initialize > save > saved to device storage or RAM > read > data can be reused.

Storage capacity: 1 MB per file for keystore.external.local, 850 KB for keystore.internal.local

Keys that can be saved: 20 for keystore.external.local, 15 for keystore.internal.local, 27 for keystore.session.store

Files that can be created: 3 for keystore.external.local, 2 for keystore.internal.local

Compatibility: Java 7 or later
